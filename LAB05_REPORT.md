What does HEAD represent?

HEAD is a marker that shows which branch or commit you are currently working on. When you switch branches, HEAD moves to the latest commit of that branch



Under what conditions does a merge conflict occur?

A merge conflict happens when you try to merge two branches that have different changes in the same line of the same file. Git doesn't know which version to keep, so it asks the user to fix it manually



What is the difference between git clone and git pull?

git clone copies whole repository to your computer and used at the start of work, git pull copies new changes from repository to your computer
